# Guardian AI — Dual-Signal Sports Media Protection

## Google Solution Challenge 2026

Guardian AI protects sports organizations' digital assets using a 
dual-signal architecture:

- **Signal A**: Invisible DCT frequency-domain watermark (invisible-watermark)
- **Signal B**: Neural content fingerprint via Google Vision API + FAISS

## Tech Stack
- Frontend: React + JavaScript (Vite) + Tailwind CSS
- Backend: Python 3.11 + FastAPI + Google Cloud Run
- Database: Firebase Firestore
- AI: Gemini 1.5 Pro + Google Vision API
